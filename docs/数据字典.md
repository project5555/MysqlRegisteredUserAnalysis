# 用户注册数据字典

## 数据表说明

### 表名: `user`

**描述**: 存储用户注册信息和基本登录数据

**用途**: 用户注册行为分析、用户增长趋势分析

**更新时间**: 2017-01-01 至 2019-12-31

---

## 字段详情

### 1. username
- **字段名**: username
- **数据类型**: varchar(255)
- **是否可空**: YES (DEFAULT NULL)
- **描述**: 用户唯一标识符
- **示例值**: mr000001, mr000002
- **说明**: 系统自动生成的用户编号，格式为"mr" + 6位数字
- **分析用途**: 
  - 用户唯一识别
  - 用户数量统计
  - 用户增长趋势分析

### 2. last_login_time
- **字段名**: last_login_time
- **数据类型**: varchar(255)
- **是否可空**: YES (DEFAULT NULL)
- **描述**: 用户最后登录时间
- **示例值**: 2017/01/01 1:57, 2017/01/01 7:33
- **格式**: YYYY/MM/DD HH:MM
- **说明**: 记录用户最后一次登录系统的时间
- **分析用途**:
  - 用户活跃度分析
  - 用户留存分析
  - 登录行为模式分析

### 3. login_count
- **字段名**: login_count
- **数据类型**: varchar(255)
- **是否可空**: YES (DEFAULT NULL)
- **描述**: 用户登录次数
- **示例值**: 0
- **说明**: 记录用户累计登录系统的次数
- **分析用途**:
  - 用户活跃度评估
  - 用户粘性分析
  - 用户价值分层

### 4. addtime
- **字段名**: addtime
- **数据类型**: varchar(255)
- **是否可空**: YES (DEFAULT NULL)
- **描述**: 用户注册时间
- **示例值**: 2017/01/01 1:57, 2017/01/01 7:33
- **格式**: YYYY/MM/DD HH:MM
- **说明**: 记录用户注册账号的时间，是用户增长分析的核心字段
- **分析用途**:
  - **核心字段**: 用户注册时间序列分析
  - 用户增长趋势分析
  - 注册高峰期识别
  - 同期群分析(Cohort Analysis)

---

## 数据质量说明

### 数据完整性
- **记录总数**: 约3年用户注册数据（2017-2019）
- **字段完整性**: 
  - username: 100%（非空）
  - addtime: 100%（非空）
  - last_login_time: 可能为空
  - login_count: 可能为空或初始值为0

### 数据质量问题
1. **时间格式**: 存储为varchar格式，需要转换为datetime类型进行分析
2. **时区问题**: 未明确时区，默认为系统时区
3. **数据精度**: 时间只精确到分钟，无秒级数据

### 数据清洗需求
1. **时间格式转换**: 将varchar转换为datetime类型
2. **缺失值处理**: 处理last_login_time和login_count的缺失值
3. **异常值检测**: 检查是否存在异常的时间数据

---

## 分析维度

### 时间维度
- **年度**: 2017年、2018年、2019年
- **季度**: Q1、Q2、Q3、Q4
- **月度**: 1-12月
- **周度**: 周一到周日
- **日度**: 具体日期
- **小时**: 0-23时

### 用户维度
- **用户ID**: username唯一标识
- **注册时间**: addtime
- **活跃度**: login_count
- **最后活跃**: last_login_time

## SQL查询示例

### 1. 查询年度注册用户数
```sql
SELECT 
    YEAR(STR_TO_DATE(addtime, '%Y/%m/%d %H:%i')) as year,
    COUNT(*) as user_count
FROM user
GROUP BY YEAR(STR_TO_DATE(addtime, '%Y/%m/%d %H:%i'))
ORDER BY year;
```

### 2. 查询月度注册趋势
```sql
SELECT 
    DATE_FORMAT(STR_TO_DATE(addtime, '%Y/%m/%d %H:%i'), '%Y-%m') as month,
    COUNT(*) as user_count
FROM user
GROUP BY DATE_FORMAT(STR_TO_DATE(addtime, '%Y/%m/%d %H:%i'), '%Y-%m')
ORDER BY month;
```

### 3. 查询日注册分布
```sql
SELECT 
    DATE(STR_TO_DATE(addtime, '%Y/%m/%d %H:%i')) as date,
    COUNT(*) as user_count
FROM user
WHERE STR_TO_DATE(addtime, '%Y/%m/%d %H:%i') BETWEEN '2018-04-01' AND '2018-04-30'
GROUP BY DATE(STR_TO_DATE(addtime, '%Y/%m/%d %H:%i'))
ORDER BY date;
```

### 4. 查询小时注册分布
```sql
SELECT 
    HOUR(STR_TO_DATE(addtime, '%Y/%m/%d %H:%i')) as hour,
    COUNT(*) as user_count
FROM user
GROUP BY HOUR(STR_TO_DATE(addtime, '%Y/%m/%d %H:%i'))
ORDER BY hour;
```
